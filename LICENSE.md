## Intended public-release license

When the repository is opened up for the accompanying blog publication, the intent is a dual-license posture matching Learning Commons' own:

- **Code** (everything under `src/`, `notebooks/`, scripts) — MIT License
- **Data, results, methodology docs, prose** (everything under `docs/`, `data/processed/`, `data/results/`) — Creative Commons Attribution 4.0 International (CC BY 4.0)

This decision is recorded so contributors understand the eventual destination from day one. The dual posture keeps the code freely vendorable and the writeups citable, with the same attribution norms that apply to the Knowledge Graph and Evaluators we build on.

Final license text will be written in at the moment of public release.
